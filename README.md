# SpaceX Falcon 9 Landing Prediction## AuthorAntonio Martinez---## Project OverviewThis project analyzes SpaceX Falcon 9 launch data to predict whether the first stage of a rocket will land successfully. SpaceX has significantly reduced launch costs through reusable rocket technology, making landing success a critical factor in mission efficiency.The project integrates data from multiple sources, performs exploratory analysis, and develops machine learning models to predict landing outcomes.---## Objectives- Identify key factors influencing landing success  - Perform exploratory data analysis (EDA) to uncover trends  - Build and evaluate classification models  - Develop interactive visualizations and dashboards  ---## Data Sources- SpaceX REST API for launch, payload, and booster data  - Wikipedia for historical launch records using web scraping  ---## Data Processing- Cleaned and handled missing and inconsistent values  - Converted landing outcomes into binary labels:  - 1 = Successful landing    - 0 = Failed landing  - Standardized numerical features for machine learning models  ---## Exploratory Data AnalysisExploratory Data Analysis was conducted using both visualization and SQL techniques to identify trends and relationships in the data.### Key Findings- Launch success increases with flight number, indicating an experience effect  - Lower payload masses are generally associated with higher success rates  - Launch success rates have improved consistently over time  - Certain launch sites demonstrate higher reliability  ---## SQL AnalysisSQL queries were used to:- Identify unique launch sites  - Filter launches based on specific conditions  - Calculate payload statistics  - Compare successful and failed mission outcomes  ---## Interactive Visualizations### Folium Map- Visualized launch site locations  - Added markers and geographic context to better understand spatial distribution  ### Plotly Dash Dashboard- Implemented an interactive dropdown for launch site selection  - Added payload range filtering  - Displayed success distribution using a pie chart  - Visualized payload versus outcome using a scatter plot  ---## Machine Learning ModelsThe following classification models were developed and evaluated:- Logistic Regression  - Support Vector Machine (SVM)  - Decision Tree  - K-Nearest Neighbors (KNN)  ### Model Performance| Model | Accuracy ||------|----------|| Logistic Regression | 83.3% || SVM | 83.3% || Decision Tree | 88.9% || KNN | 83.3% |### Best Performing ModelThe Decision Tree model achieved the highest accuracy. Its performance suggests that nonlinear relationships between features play an important role in predicting landing success.---## Key Takeaways- Landing success is influenced by flight experience, payload mass, and launch site  - SpaceX has demonstrated continuous improvement in launch reliability  - Machine learning models can effectively predict landing outcomes  ---## Repository Contents- Data collection notebooks (API and web scraping)  - Data wrangling and preprocessing  - Exploratory Data Analysis (visualization and SQL)  - Plotly Dash dashboard (Python script)  - Machine learning notebook  - Markdown documentation  - Final presentation  ---## Technologies Used- Python  - Pandas and NumPy  - Matplotlib and Seaborn  - SQL  - Plotly Dash  - Folium  - Scikit-learn  ---## Future Work- Incorporate additional data sources to improve model robustness  - Explore ensemble modeling techniques to increase predictive performance  - Deploy the dashboard as a web-based application  ---## NotesThis project was completed as part of the IBM Data Science Professional Certificate Capstone.
